# プロジェクト名
心理テスト

##
自分で初めて作るwebアプリ。
複数のモードで遊べる。
始めてちゃんと作るGitHubプロジェクト。

##
- 機能1：単純な心理テスト（他のサイトを参考）
- 機能2：管理者が作成した心理テスト
- 機能3：使用者が答えた質問を反映する物語が読める
- 機能4：性格診断
- 機能5：管理者がテーマを考えたワードウルフ
- 機能6：管理者が作成した適職診断

##
使い方
1. 遊びたいモードを選択する
2. ホームに戻りたい時はホームボタンを押す
