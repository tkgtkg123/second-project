## 使用言語
 - JavaScript 
 - html
 - css、SCSS

## 起動方法（ローカル）
1. このリポジトリをクローンする
2. index.html をブラウザで開く

## 設計意図
各モードで遊んでいる時にユーザーが最小限の操作でホーム画面に移動・ログアウトができるように、
ホームボタン、ログアウトボタンを上部固定にした。
ホーム画面は各機能が一覧で見ることができるレイアウトとした。
各機能のタイトルは一目でイメージできず好奇心を刺激するようなタイトルとした。
質問に回答していくシステムはユーザーが主体的に参加している感覚になりやすく、
サイトに没入感を抱きやすいと考えたため質問回答型の機能を複数設計した。
