# プロジェクト名
心理テスト

## 機能
- 心理テスト
- 質問の回答によってストーリーが変わる物語
- 性格診断
- ワードウルフ
- 適職診断

## 使用言語
 - JavaScript 
 - html
 - css、SCSS

## 起動方法（ローカル）
1. このリポジトリをクローンする
2. index.html をブラウザで開く

## 設計意図
本アプリは質問形式の心理テストを通じてユーザーのタイプ判定を行う簡易Webアプリです。
Web制作の基礎（デザイン → 質問ロジック → 結果表示 → ページ遷移）を一気通貫で構築することを目的としています。
また、UIの改善やアクセシビリティ対応、スコアロジックのリファクタリングなど、プロダクトの改修プロセスをREADMEで明示し、成長の軌跡を可視化します。
